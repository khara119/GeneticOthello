# What is this?

CUIでただただ眺めるだけのCPU vs CPUのオセロゲームです。

ただコマを返されにくいところに優先的に置いていくことを学習して
コマを置いていくので、そんなに良い頭ではないです。

# How to play?

1. `javac *.java`で全部コンパイルします。
1. `java Main`で実行します。

※現状各世代ごとの優秀遺伝子での対戦のみ出力します。
　各世代の各遺伝子同士の対戦も見たければ適度なところに標準出力入れましょう。
